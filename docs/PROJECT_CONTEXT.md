## Сториттелинг (12 экранов)

### 1. Intro — интрига
- **Контент:** "Эй, у нас тут кое-что намечается..."
- **Визуал:** Рамка из веток, имена крупно
- **Эффект:** Золотые огоньки (fireflies)
- **Цель:** Интрига без слова "свадьба"

### 2. Февраль 2023 — "точно нет"
- **Контент:** Она смотрит на его фото скептически
- **Визуал:** Картинка генерация — девушка смотрит на телефон с сомнением
- **Подпись:** "точно нет" / "not my type"
- **Цель:** Показать, что начало было не романтичным

### 3. 9 декабря 2023 — первая встреча
- **Контент:** "ещё до привет она всё поняла"
- **Визуал:** Он входит в дверь, она смотрит
- **Цель:** Момент "щелчка"

### 4. Минуту спустя — неловкое рукопожатие
- **Контент:** Столкнулись руками (highfive вместо рукопожатия)
- **Визуал:** highfive.png (уже есть)
- **Цель:** Юмор, неловкость

### 5. Смех — оба смеются
- **Контент:** "и тут оба рассмеялись"
- **Визуал:** Оба смеются от неловкости
- **Эффект:** Canvas искры/блики
- **Цель:** Момент connection

### 6. Вместе — "ни дня без общения"
- **Контент:** Статистика или просто фраза
- **Визуал:** Абстрактно или фото-коллаж стиль
- **Цель:** Показать период отношений

### 7. Предложение — зимняя сцена
- **Контент:** "Она сказала да" / декабрь 2024
- **Визуал:** предложение.png (уже есть) — он на колене, домик, огоньки
- **Эффект:** Падающий снег
- **Цель:** Кульминация love story

### 8. Площадка — Forest Dew
- **Контент:** Showcase площадки
- **Визуал:** Фото/рендер площадки или генерация
- **Цель:** Показать место

### 9. Приглашение — "Свои люди"
- **Контент:** "Мы зовём только своих людей. И ты в списке."
- **Визуал:** Минималистично, текст
- **Цель:** Тёплое приглашение

### 10. Подготовка — практическая инфа
- **Дресс-код:** (уточнить)
- **Подарки:** "Лучший подарок — вклад в нашу квартиру" (фонд)
- **Ограничения:** 16+, можно +1
- **Цель:** Вся практическая информация

### 11. Шпаргалка — один экран для скриншота
- **Контент:** Дата, время, адрес, QR на карту, контакты
- **Визуал:** Компактно, всё важное
- **Цель:** Гость делает скриншот и всё под рукой

### 12. RSVP — манипулятивная форма
- **Контент:** Шкала 1-10 "насколько ты счастлив за нас"
- **Кнопки:** "Буду!" (большая) / "Не смогу" (маленькая, грустная)
- **Реакции:** Разные ответы на разные выборы
- **Дедлайн:** 1 мая 2026
- **Цель:** Юмор + сбор ответов

## Технические наработки

### Что работает хорошо
```
Canvas 2D + radialGradient = настоящий glow на огоньках
object-fit: cover = картинка на весь экран
text-shadow + gradient overlay = читаемый текст на фото
100dvh = правильная высота на мобилках
scroll-snap = плавное переключение экранов
```

### Что не работает / сложно
```
Three.js pointsMaterial без шейдеров = квадратики
Генерация PNG слоёв с прозрачностью = ореолы от хромакея
Сложные GLSL шейдеры = нестабильно на мобилках
```

### Рекомендуемый стек
```
React + Vite
Tailwind CSS
Canvas 2D для частиц (огоньки, снег, искры)
CSS transitions для переходов между экранами
Framer Motion для анимации элементов (опционально)
```

### Структура проекта
```
/public
  /images
    frame_clean.png     — рамка из веток (прозрачный фон)
    proposal.png        — сцена предложения
    [другие картинки по мере генерации]

/src
  App.jsx              — основной компонент
  index.css            — стили + Tailwind
  main.jsx             — entry point
  /components
    IntroScreen.jsx
    ProposalScreen.jsx
    RSVPScreen.jsx
    [другие экраны]
```

## Готовые ассеты

### Изображения
1. `frame_clean.png` — ботаническая рамка из веток (прозрачный центр)
2. `предложение.png` — сцена предложения руки и сердца (оригинал, полная картинка)

### Для генерации нужно
- Экран 2: Она смотрит на телефон скептически
- Экран 3: Первая встреча (он входит)
- Экран 5: Оба смеются
- Экран 6: "Вместе" — абстракция или коллаж
- Экран 8: Площадка Forest Dew

### Промпт-стратегия для генерации
- Использовать магента (#FF00FF) фон для объектов, требующих прозрачности
- Полные сцены генерировать без прозрачности (как proposal.png)
- Удалять фон через: `is_magenta = (r > 150) & (g < 120) & (b > 150)`
